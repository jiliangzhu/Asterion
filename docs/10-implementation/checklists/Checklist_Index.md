## 4. 目录规则

- 当前阶段需要关心“当前 planning 入口是什么”时，先看本目录。
- 若只是回看历史 accepted 阶段，再看本目录中的 `historical accepted` 区。
- 需要跨阶段历史结论或 AlphaDesk 迁移背景时，不要优先读 archive checklist，先看 migration ledger。
- 若某份 checklist 已不再承担 active truth-source 角色，应改成 archive note，而不是继续和 active closeout checklist 混放为同等级入口。
